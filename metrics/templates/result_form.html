Pre/Post Test: Confronto prima e dopo un cambiamento

  • Before: Valore prima del rilascio/cambiamento
  • After: Valore dopo il rilascio/cambiamento
  • Delta %: (After - Before) / Before × 100
  • Decisione: Persevere se delta ≥ target ({{ indicator.target_uplift }}%), altrimenti Pivot
    • {% if indicator.role == 'guardrail' %}
  • Guardrail: Deve rimanere stabile (delta > -5%)
    • {% endif %}
